# KAG 项目二度开发说明
本内容基于 [OpenSPG/KAG](https://github.com/OpenSPG/KAG) 项目进行二度开发，核心信息如下：

## 1. 开发手册
KAG 官方开发手册地址：<https://openspg.github.io/v2/docs_ch>

## 2. 系统设计
系统设计相关文档可查看项目内路径：`api/docs`

## 3. 接口开发详情
### 3.1 代码目录
接口代码统一存放于项目根目录下的 `api` 目录中。

### 3.2 开发框架
接口开发基于 **FastAPI** 框架实现。

### 3.3 核心接口功能
接口主要实现对 OpenSPG 项目的外部操控能力，具体包含：
- 创建项目
- 初始化项目
- 文件上传
- 图数据库生成
- 数据检索
